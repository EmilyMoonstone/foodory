# Core User Flows

## 1) Onboarding

1. User signs up or logs in.
2. User creates profile.
3. User adds friends and/or joins groups.

## 2) Create a post

1. User opens composer.
2. User adds image.
3. User fills fields:
   - work level
   - taste
   - recipe link
   - text
4. User optionally adds:
   - location / restaurant mode
   - cooked-together user tags
5. User selects audience:
   - all friends
   - one or multiple groups
   - both
6. User publishes post.

## 3) View group content with rules

### Daily unlock groups

1. User opens group feed.
2. If user has not posted today, others' posts for today are locked.
3. After user posts today, today's group posts become visible.

### Sunday reveal groups

1. User opens group feed during week.
2. Current week posts are hidden.
3. On Sunday, weekly posts unlock for the group.

## 4) Interact and curate

1. User reacts to posts.
2. User saves posts.
3. Saved posts can be added to collaborative cookbooks.

## 5) Explore history and recap

1. User opens personal history to browse own past posts.
2. User opens group history to browse prior group posts (with rule-aware visibility).
3. User views weekly group recap story generated from top engagement and reactions.
